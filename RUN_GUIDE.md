# دليل تشغيل مشروع "رحلتي" 🌍

لضمان عمل التطبيق وربطه بالبيانات بشكل سليم، اتبع الترتيب التالي:

## 1. تشغيل السيرفر (Backend)
- افتح التيرمينال في مجلد `backend`
- اكتب: `npm run dev`
- تأكد من ظهور رسالة النجاح والاتصال بقاعدة البيانات.

## 2. تشغيل المحاكي (Android Emulator)
- افتح Android Studio.
- قم بتشغيل جهاز الأندرويد الافتراضي من الـ Device Manager.

## 3. تشغيل تطبيق الموبايل (Flutter)
- افتح التيرمينال في مجلد `flutter`
- اكتب: `flutter run`

## 4. أوامر مفيدة أثناء العمل (داخل Flutter Terminal)
- اضغط `r`: لعمل Hot Reload (تحديث سريع للتغييرات في التصميم).
- اضغط `R`: لعمل Hot Restart (إعادة بناء الواجهة بالكامل - مفيد عند تغيير الـ Providers أو العناوين).
- اضغط `q`: للخروج وإيقاف التطبيق.
